## The sizeof Operator

`sizeof` is a compile-time operator that returns the size in bytes of a type or expression. It is evaluated by the compiler, not at runtime.

## Pointer Size Rule

From the slides:
> "The size of a pointer is always 8 bytes in a 64-bit architecture independent of the type of the pointer."

All pointer types are the same size — 8 bytes on a 64-bit system — regardless of what they point to:

```
sizeof(int*)   == 8 bytes
sizeof(char*)  == 8 bytes
sizeof(long*)  == 8 bytes
sizeof(void*)  == 8 bytes
```

This is because a pointer holds a virtual address, and on a 64-bit system all addresses are 64 bits (8 bytes).

## Type Sizes (from slides)

| Type | Size | sizeof |
|---|---|---|
| `char` | 1 byte | `sizeof(char) == 1` |
| `int` | 4 bytes | `sizeof(int) == 4` |
| `long` | 8 bytes | `sizeof(long) == 8` |
| `int*` | 8 bytes | `sizeof(int*) == 8` |
| `char*` | 8 bytes | `sizeof(char*) == 8` |

From the slides:
```
sizeof(int)  == 4 bytes
sizeof(long) == 8 bytes
sizeof(char) == 1 byte

```

## Using sizeof in Allocations

Always use `sizeof` in `malloc` calls rather than hard-coding sizes:

```c
int *p = (int*)malloc(sizeof(int));         // allocate one int
int *arr = (int*)malloc(10 * sizeof(int));  // allocate 10 ints
```

This makes the code portable across platforms where type sizes may differ.

## sizeof and Arrays

For a statically declared array, `sizeof` gives the total size:

```c
int a[20];
sizeof(a)        // == 80 (20 * sizeof(int) = 20 * 4)
sizeof(a[0])     // == 4 (size of one element)
sizeof(a) / sizeof(a[0])  // == 20 (number of elements)
```

**Warning**: Once an array decays to a pointer (passed to a function), `sizeof` gives the pointer size (8), not the array size.

## Key Terms

| Term | Meaning |
|---|---|
| `sizeof(type)` | Compile-time size of a type in bytes |
| Pointer size | Always 8 bytes on 64-bit; independent of pointed-to type |
| `sizeof(int)` | 4 bytes |
| `sizeof(long)` | 8 bytes |
| `sizeof(char)` | 1 byte |
| Array decay | Array passed to function becomes a pointer; `sizeof` gives 8, not array size |
