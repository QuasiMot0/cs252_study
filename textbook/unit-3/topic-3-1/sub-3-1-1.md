## What Is a Pointer?

From the slides:
> "A pointer is a variable that contains an address in memory."
> "In a 64-bit architecture, the size of a pointer is 8 bytes independent on the type of the pointer."

A pointer stores the address of another variable. The type of the pointer tells the compiler how to interpret the bytes at that address (how many bytes, how to do arithmetic), but all pointer variables themselves are 8 bytes on a 64-bit system.

```c
char c = 'A';      // c is at some address, say 12; contains 65 (ASCII 'A')
char *p = &c;      // p is at address 20; contains 12 (the address of c)
// p:20 → 12,  c:12 → 65
```

## The `*` Operator (Dereference)

`*p` means "go to the address stored in p and read/write what's there."

```c
char c = 'A';
char *p = &c;

printf("%c\n", *p);  // prints 'A' — reads c through p
*p = 'B';            // writes 'B' into c through p
```

In a declaration, `*` means "this is a pointer to":
```c
int *p;       // p is a pointer to int
char *s;      // s is a pointer to char
```

## The `&` Operator (Address-Of)

`&x` returns the address of variable `x`:

```c
int x = 42;
int *p = &x;   // p now holds the address of x
```

## NULL

`NULL` is the zero pointer — a pointer guaranteed not to point to any valid memory. It is defined as `(void*)0`. Dereferencing `NULL` causes a SEGV:

```c
int *p = NULL;
*p = 5;   // SEGV — cannot dereference NULL
```

Checking for `NULL` before dereferencing is essential defensive programming.

## Four Ways to Get a Pointer Value

From the slides:

1. **Numerical cast** (OS/kernel/embedded only):
   ```c
   char *p = (char *)0x1800;
   *p = 5;   // Store 5 at physical address 0x1800
   ```
   > "Assigning a numerical value to a pointer isn't recommended and only left to programmers of OS, kernels, device drivers, embedded systems."

2. **Address-of operator**:
   ```c
   int *p;
   int buff[30];
   p = &buff[1];   // p points to buff[1]
   *p = 78;
   ```

3. **Heap allocation** (`malloc`/`new`):
   ```c
   int *p = new int;
   int *q = (int*)malloc(sizeof(int));
   *q = 5;   // &q==5000, q==1000, *q==5
   ```

4. **Function parameter** (pass to modify):
   ```c
   void swap(int *a, int *b) {
       int temp = *a;
       *a = *b;
       *b = temp;
   }
   // In main:
   swap(&x, &y);
   ```

## Casting to Overlay Struct onto Buffer

From the slides:
> "You can store a variable of any type in memory by using casts."

```c
typedef struct {
    unsigned long size;
    unsigned long leftSize;
} ObjectHeader;

char buf[1000];
ObjectHeader *hdr = (ObjectHeader *)buf;
hdr->size = objectSize;
hdr->leftSize = leftSize;
```

This treats the raw bytes of `buf` as an `ObjectHeader` without copying.

## Key Terms

| Term | Meaning |
|---|---|
| Pointer | Variable that holds a memory address; always 8 bytes on 64-bit |
| `*p` (dereference) | Read or write the value at the address stored in `p` |
| `&x` (address-of) | Returns the address of variable `x` |
| `NULL` | Zero pointer; dereferencing it causes SEGV |
| Cast overlay | Treating raw memory as a struct via `(StructType*)buffer` |
