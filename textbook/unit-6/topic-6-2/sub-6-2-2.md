## The open() System Call

From the slides:
> "`open(filename, mode, [permissions])` — opens a file and returns a file descriptor."

```c
int fd = open(filename, mode);           // open existing file
int fd = open(filename, mode, perms);    // create new file (requires O_CREAT)
```

Returns a **file descriptor** (integer ≥ 0) on success, or **-1** on error.

## open() Flags

Flags are combined with bitwise OR (`|`):

| Flag | Meaning |
|---|---|
| `O_RDONLY` | Open for reading only |
| `O_WRONLY` | Open for writing only |
| `O_RDWR` | Open for reading and writing |
| `O_CREAT` | Create the file if it does not exist (requires permissions argument) |
| `O_APPEND` | All writes go to the end of the file; offset always set to EOF before write |
| `O_TRUNC` | Truncate file to length 0 on open (discard existing contents) |

## Examples

```c
#include <fcntl.h>

// Read an existing file
int fd = open("data.txt", O_RDONLY);

// Write to a file; create if not present, truncate if it exists
int fd = open("out.txt", O_WRONLY | O_CREAT | O_TRUNC, 0644);

// Append to a log file
int fd = open("log.txt", O_WRONLY | O_CREAT | O_APPEND, 0644);

// Read and write
int fd = open("file.bin", O_RDWR);
```

## Closing a File Descriptor

```c
close(fd);
```

`close()` decrements the **reference count** of the Open File Object. When the reference count reaches 0, the system releases the object and frees the offset tracking state.

## Relationship to the Open File Table

When `open()` is called:
1. The kernel creates a new Open File Object (inode reference, mode, offset = 0, ref count = 1).
2. A new entry is inserted into the process's per-process fd table.
3. The fd integer (index) is returned to the caller.

`O_APPEND` causes the offset to jump to EOF before every `write()`, ensuring multiple writers do not overwrite each other.

## Key Terms

| Term | Meaning |
|---|---|
| `open(file, flags)` | System call to open a file; returns fd or -1 |
| `O_RDONLY` | Read-only access |
| `O_WRONLY` | Write-only access |
| `O_RDWR` | Read and write access |
| `O_CREAT` | Create file if it doesn't exist; requires permissions argument |
| `O_TRUNC` | Truncate file to zero length on open |
| `O_APPEND` | Set offset to EOF before every write |
| `close(fd)` | Decrement ref count; release file object when count reaches 0 |
