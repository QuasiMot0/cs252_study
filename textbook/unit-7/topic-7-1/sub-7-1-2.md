## if / then / else / fi

From the slides (hello-loop.sh):

```bash
if [ $# -lt 1 ]
then
  echo
  echo "$0 needs at least one argument"
  echo "  Eg."
  echo "      $0 Mickey Donald Daisy"
  exit 1
fi
```

Structure:
```bash
if [ condition ]
then
  # commands if true
else
  # commands if false
fi
```

- `[` and `]` are the `test` command — spaces inside are required.
- `fi` closes the `if` block.
- `elif` can be used for additional conditions.

## for Loops

From the slides (hello-loop.sh):

```bash
for friend in $*
do
  echo "Hello $friend"
done
```

From the slides (count-files.sh), nested for loops:

```bash
for dir in $dirs
do
  echo $dir:
  for file in $dir/*
  do
    echo "$count: $file"
    count=`expr $count + 1`
  done
done
```

General form:
```bash
for variable in list
do
  # commands
done
```

`$*` expands to all positional arguments as a single word list. `$dir/*` uses glob expansion to list all files in `$dir`.

## while Loops

```bash
while [ condition ]
do
  # commands
done
```

Example — loop every 10 seconds (from Lab 2 procinfo.sh):
```bash
while true
do
  # print process info
  sleep 10
done
```

## exit Codes

From the slides:
```bash
exit 1    # non-zero = failure
exit 0    # zero = success
```

`exit N` terminates the script with exit code N. By convention, 0 means success and any non-zero value means an error. The exit code of the last command run is stored in `$?`.

## Complete Example: count-files.sh

From the slides:

```bash
#!/bin/bash
# Counts how many files are in the directories passed as parameters.
# If no directories are passed, uses the current directory.

if [ $# -lt 1 ]
then
  dirs=.
else
  dirs=$*
fi

count=0

for dir in $dirs
do
  echo $dir:
  for file in $dir/*
  do
    echo "$count: $file"
    count=`expr $count + 1`
  done
done

echo "$count files found"
```

## Key Terms

| Term | Meaning |
|---|---|
| `if [ cond ]` | Test condition; spaces inside `[ ]` are required |
| `then` / `else` / `fi` | if-block keywords |
| `for var in list` / `do` / `done` | For-loop iterating over a list |
| `while [ cond ]` / `do` / `done` | While-loop |
| `$*` | All positional arguments as a word list |
| `exit N` | Exit script with code N (0=success, non-zero=error) |
| `$?` | Exit code of the last command |
| `expr A + B` | Integer arithmetic (result printed to stdout) |
