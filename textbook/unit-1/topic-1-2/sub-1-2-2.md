## Addressing Modes in x86-64

An **addressing mode** determines how an instruction specifies its operands. x86-64 supports four main modes.

From the slides, these correspond to different ways to provide a value to an instruction like `movq src, dst`:

## 1. Immediate

A constant literal encoded directly in the instruction.

```asm
movq $0x501208, %rdi   # Load constant 0x501208 into %rdi
movq $42, %rax         # Load 42 into %rax
```

- Prefix: `$` before the value
- The value is embedded in the instruction itself

## 2. Direct Register

Copy a value from one register to another.

```asm
movq %rax, %rdi   # %rdi = %rax
```

- No `$` or `()` — just a register name

## 3. Indirect (Register Indirect)

Dereference the address held in a register. The register holds a **pointer**; the instruction reads/writes the memory at that address.

```asm
movq (%rdi), %rsi   # %rsi = *(%rdi)  — read 8 bytes at address in %rdi
movq %rax, (%rdi)   # *(%rdi) = %rax  — write %rax to address in %rdi
```

- Syntax: `(%register)` — parentheses mean "treat as pointer"
- Equivalent to `*ptr` in C

## 4. Direct Memory (Absolute Address)

Access a specific absolute memory address, possibly with a register offset.

```asm
movq 0x501208(%rdi), %rdi   # %rdi = *(0x501208 + %rdi)
```

- Used to access global variables and struct fields
- The offset is a constant; the register provides a base

## C-Type Sizes and Suffixes

| C Type | Size | Suffix |
|---|---|---|
| `char` | 1 byte | `b` |
| `short` | 2 bytes | `w` |
| `int`, `unsigned` | 4 bytes | `l` |
| `long`, `unsigned long`, `char*` | 8 bytes | `q` |
| `float` | 4 bytes | `s` |
| `double` | 8 bytes | `d` |

The `q` suffix (quad word = 64 bits) is used throughout most CS252 examples: `movq`, `addq`, `subq`, etc.

## Key Terms

| Term | Meaning |
|---|---|
| Immediate | Constant encoded in the instruction; prefixed with `$` |
| Direct register | Source/destination is a register |
| Indirect | Register contains a pointer; `(%reg)` dereferences it |
| Direct memory | Absolute or offset address: `offset(%reg)` |
| `q` suffix | Quad word — 64-bit operand size |
