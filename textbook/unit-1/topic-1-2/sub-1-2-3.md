## Overview

All x86-64 instructions that operate on 64-bit values use the `q` suffix (quad word). The slides note:
> "'q' suffix = quad word / 64-bit; no penalty for 64-bit operations."

## Data Movement Instructions

| Instruction | Effect | Notes |
|---|---|---|
| `movq S, D` | `D = S` | Most common; 64-bit move |
| `movabsq I, R` | `R = I` | Move 64-bit absolute immediate to register |
| `movslq S, D` | `D = sign_extend_32(S)` | Sign-extend 32-bit → 64-bit |
| `movsbq S, D` | `D = sign_extend_8(S)` | Sign-extend 8-bit → 64-bit |
| `movzbq S, D` | `D = zero_extend_8(S)` | Zero-extend 8-bit → 64-bit |
| `pushq S` | `rsp -= 8; *rsp = S` | Push 64-bit value onto stack |
| `popq D` | `D = *rsp; rsp += 8` | Pop 64-bit value from stack |
| `leaq S, D` | `D = address_of(S)` | Load effective address (no memory read) |

## Arithmetic Instructions

| Instruction | Effect |
|---|---|
| `addq S, D` | `D += S` |
| `subq S, D` | `D -= S` |
| `imulq S, D` | `D *= S` (signed) |
| `incq D` | `D++` |
| `decq D` | `D--` |
| `negq D` | `D = -D` |

## Bitwise Instructions

| Instruction | Effect |
|---|---|
| `xorq S, D` | `D ^= S` |
| `orq S, D` | `D \|= S` |
| `andq S, D` | `D &= S` |
| `notq D` | `D = ~D` |
| `salq/shlq S, D` | `D <<= S` (left shift) |
| `sarq S, D` | `D >>= S` (arithmetic right shift, sign-preserving) |
| `shrq S, D` | `D >>= S` (logical right shift, zero-fill) |

## Comparison Instructions (Set Flags, No Result)

| Instruction | Sets Flags Based On |
|---|---|
| `cmpq S2, S1` | `S1 - S2` (does not store result) |
| `testq S2, S1` | `S1 & S2` (does not store result) |

## leaq — Load Effective Address

`leaq` computes an address but does **not** dereference it. Often used for arithmetic tricks:

```asm
leaq (%rdi, %rsi, 8), %rax   # %rax = %rdi + 8*%rsi  (address calculation)
leaq 5(%rdi), %rax            # %rax = %rdi + 5
```

## Key Terms

| Term | Meaning |
|---|---|
| `movq` | 64-bit register/memory move |
| `leaq` | Computes address (pointer arithmetic) without dereferencing |
| `pushq`/`popq` | Push/pop 64-bit value; adjusts `%rsp` by 8 |
| `cmpq S2,S1` | Computes `S1-S2`, sets flags; use before conditional jump |
| `testq S2,S1` | Computes `S1&S2`, sets flags; commonly `testq %rax,%rax` to check zero |
| `sarq` vs `shrq` | Arithmetic vs logical right shift (sign-preserving vs zero-fill) |
