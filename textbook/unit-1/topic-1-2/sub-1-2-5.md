## General Pattern

The compiler translates C control flow into conditional jumps and labels in assembly. The key insight is that high-level control structures (if/else, while, for) become `cmpq` + `jXX` + `jmp` sequences.

## Translating an if/else Statement

**C source:**
```c
long max(long a, long b) {
    if (a > b)
        return a;
    else
        return b;
}
```

**Assembly:**
```asm
max:
    pushq  %rbp
    movq   %rsp, %rbp
    cmpq   %rsi, %rdi   # compare a (%rdi) vs b (%rsi): a - b
    jle    else_branch  # if a <= b, jump to else
    movq   %rdi, %rax   # then: return a
    jmp    end_max
else_branch:
    movq   %rsi, %rax   # else: return b
end_max:
    leave
    ret
```

The compiler tests the **inverse** condition and jumps to the else-branch. The then-branch falls through naturally.

## Translating a while Loop

**C source:**
```c
long maxarray(long *a, long n) {
    long max = a[0];
    long i = 1;
    while (i < n) {
        if (a[i] > max)
            max = a[i];
        i++;
    }
    return max;
}
```

**Assembly pattern:**
```asm
    # ... setup ...
while_label:
    cmpq  %rdx, %rdi   # compare i vs n
    jle   afterw       # if i >= n, exit loop
    # ... inner if body ...
    addq  $1, %rdi     # i++
    jmp   while_label  # loop back
afterw:
    leave
    ret
```

## Array Dereferencing

To access `a[i]` where `a` is a `long*` (8 bytes per element):

```asm
movq  %rdx, %rcx       # copy i
imulq $8, %rcx         # multiply by element size (8)
addq  %rsi, %rcx       # add base address: &a[i] = base + 8*i
movq  (%rcx), %rax     # dereference: load a[i]
```

Or using the scaled-index addressing mode:

```asm
movq  (%rsi, %rdx, 8), %rax   # load a[i] directly
```

## Key Terms

| Term | Meaning |
|---|---|
| Inverse condition | Compiler tests opposite condition and jumps to else; then-branch falls through |
| Label | Named point in code; target of jump instructions |
| `jmp label` | Unconditional jump; used at end of then-branch to skip else |
| Scaled index | `(%base, %index, scale)` addressing for array access: `base + scale * index` |
| `leave` | Restore `%rsp` from `%rbp` and pop `%rbp`; shorthand for function epilogue |
| `ret` | Return to caller; pops return address from stack into `%rip` |
