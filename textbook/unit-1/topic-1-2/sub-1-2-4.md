## The Flags Register

The **flags register** (RFLAGS) is a special register whose individual bits are set or cleared as a side effect of arithmetic and comparison instructions.

From the slides, the relevant flags are:

| Flag | Name | Set When |
|---|---|---|
| `ZF` | Zero Flag | Result is zero |
| `SF` | Sign Flag | Result is negative |
| `OF` | Overflow Flag | Signed overflow occurred |
| `CF` | Carry Flag | Unsigned overflow / borrow |
| `AF` | Auxiliary Carry Flag | Carry from bit 3 to bit 4 |
| `PF` | Parity Flag | Result has even number of 1 bits |

## Setting the Flags

Two instructions set flags without producing a stored result:

```asm
cmpq  S2, S1    # computes S1 - S2, sets flags
testq S2, S1    # computes S1 & S2, sets flags
```

Common patterns:
```asm
cmpq  %rsi, %rdi   # sets flags from %rdi - %rsi
testq %rax, %rax   # checks if %rax == 0 (ZF=1 if zero)
```

## Conditional Jumps

After a `cmpq` or `testq`, a conditional jump branches based on the flags:

| Instruction | Alias | Condition | Flags |
|---|---|---|---|
| `JE` | `JZ` | Equal / Zero | `ZF=1` |
| `JNE` | `JNZ` | Not Equal / Not Zero | `ZF=0` |
| `JGE` | `JNL` | Greater or Equal (signed) | `SF=OF` |
| `JLE` | `JNG` | Less or Equal (signed) | `ZF=1` or `SF≠OF` |
| `JG` | `JNLE` | Greater (signed) | `ZF=0` and `SF=OF` |
| `JL` | `JNGE` | Less (signed) | `SF≠OF` |
| `JS` | — | Sign (negative) | `SF=1` |
| `JNS` | — | Not Sign | `SF=0` |

## Conditional Moves

Instead of branching, the CPU can use a **conditional move** to avoid branch mispredictions:

| Instruction | Condition |
|---|---|
| `cmove S, D` | Move if equal (ZF=1) |
| `cmovne S, D` | Move if not equal |
| `cmovs S, D` | Move if sign (negative) |
| `cmovns S, D` | Move if not sign |

## Example: Conditional Jump

```asm
cmpq  %rsi, %rdi   # compare %rdi and %rsi
jle   else_branch  # if %rdi <= %rsi, jump to else
movq  %rdi, %rax   # then-branch: return %rdi
jmp   end
else_branch:
movq  %rsi, %rax   # else-branch: return %rsi
end:
```

This implements `return (a > b) ? a : b`.

## Key Terms

| Term | Meaning |
|---|---|
| `ZF` | Zero flag; set when result equals zero |
| `SF` | Sign flag; set when result is negative |
| `OF` | Overflow flag; set on signed overflow |
| `cmpq S2,S1` | Sets flags from `S1-S2`; does not store result |
| `testq S2,S1` | Sets flags from `S1&S2`; `testq %rax,%rax` tests if zero |
| Conditional jump | Branches based on flag state after a compare |
